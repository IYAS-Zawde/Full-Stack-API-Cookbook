Concept 2

Anatomy of a Request

Demonstrates parameter parsing paradigms on the node Express routing container. Analyze query string arguments versus resource path IDs.

Query Parameters

Express extracts values mapped after the `?` delimiter into `req.query` object parameters. Used for searches/filtering.

Path Parameters

Express compiles wildcards inside API paths (like `/api/items/:id`) and exposes matched integers inside the `req.params` object.

Concept 4

CORS & Browser Ingress Security (Preflight)

Explore the mechanics of browser-based origins protection. CORS blocking happens at the client browser level, preventively blocking readable responses if CORS headers are absent.

Trigger Simulation Calls

Test standard CORS versus error states triggered by non-matching header properties.

Note: Standard headers like Access-Control-Allow-Origin: * enable simple connections. In production, always restrict backend origins to safe host lists.
